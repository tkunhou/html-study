JavaScript函式，固定寫法 function 函式名稱(參數名稱列表){其他任何想要的判斷式}，範例：
 function show(msg){
 console.log(msg)
    }
呼叫上面的函示方法如下：
show("hello")
show(350)
show(false)
若函式裡面有寫 return ，代表不管你函式的內容寫啥，回傳回來的值就是 return 所設定的內容
